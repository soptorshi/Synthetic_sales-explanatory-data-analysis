# Synthetic_sales-explanatory-data-analysis
• Dataset source (synthetic retail sales data)

• Objective (analyze revenue, customer behavior, discounts)

• Tools (Python, Pandas, Tableau)

• Key insights : 
  1. Electronics drive majority of revenue
  2. 30–39 age group has highest average spend
  3. Zero-discount orders generate higher order value
