# Synthetic Sales - Explanatory Data Analysis
• Dataset source: Synthetic retail sales (Kaggle)

• Objective: analyze revenue, customer behavior, discounts

• Tools: Python (Pandas, Numpy), Tableau

• Key insights: 
  1. Electronics contributes to the majority of the total revenue
  2. The 30–39 age group has the highest average spend
  3. Zero-discount orders generate higher order value
